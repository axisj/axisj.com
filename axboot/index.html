풀스택 웹 애플리케이션 개발 프레임워크

AXBoot와 함께라면 개발이 즐겁다.

AXBoot 엔진으로 더 빠르고 즐거운 웹 개발을 시작하세요.

웹 애플리케이션 개발에 필요한 핵심엔진을 모두 제공합니다

AXBoot 프레임워크는 웹 애플리케이션 개발을 더 쉽고 빠르게 할 수 있도록 레이아웃 시스템, 웹 컴포넌트, 서버 사이드 API 엔진을 제공합니다. 더이상 어렵고 반복적인 웹 개발은 그만! 빠르고 안정적인 엔터프라이즈 서비스 개발에 AXBoot를 사용하세요.

Demo Github Repository AXBoot Initialzr

특징

풀스택 프레임워크

프론트엔드

웹 애플리케이션 개발에 필수적인 17가지 이상의 웹 컴포넌트와 그리드/레이아웃 시스템을 제공합니다.

백엔드

Java와 Spring Framework기반으로 설계된 백엔드 컴포넌트들은 뛰어난 확장성과 안정성을 제공합니다.

다양한 레이아웃과 테마

레이아웃

웹 애플리케이션 개발에 가장 많이 사용되는 10가지 레이아웃을 제공합니다. 레이아웃 구성이 비슷한 경우 10분 만에 새로운 업무 화면을 개발 할 수 있습니다.

테마

시스템 전체의 룩앤필을 변경할 수 있는 테마 기능을 제공합니다. 테마는 17가지의 컴포넌트와 레이아웃 시스템의 색상, 모양을 모두 일괄적으로 변경합니다.

개발자 친화적 환경

순수 웹 기술

모든 컴포넌트는 Java, HTML5, ECMA 표준에 기반한 순수 웹 기술로 만들어졌습니다.

개방형 구조

모든 소스코드가 공개되어 있으며, 특정 기술에 종속적이지 않습니다.

개발환경의 자유

특정 IDE에 종속적이지 않습니다. Eclipse, IntelliJ, Sublime 등 개발자가 원하는 어떠한 도구라도 사용하여 개발할 수 있습니다.

생산성

미리 구성된 프로젝트

AXBoot Initialzr를 사용하여 프로젝트를 생성하면 AXBoot 기반의 미리 만들어진 프로젝트로 개발을 시작할 수 있습니다.

손쉬운 CRUD 구현

AXBoot CRUD 스캐폴딩 도구를 사용하면 테이블 역공학 기밥을 통해 완전히 동작하는 CRUD코드를 생성해 줍니다.

친숙한 접근

많은 문서들이 한글로 제공되며, 기술블로그를 통해 다양한 유스 케이스를 제공합니다.

엔터프라이즈 수준의 프레임워크

확장성

데이터베이스 풀링, 데이터 캐싱을 옵션들을 기본적으로 제공하며 대규모 엔터프라이즈 환경에서 트래픽을 안정적으로 처리하고, 병렬 확장이 가능하도록 설계되었습니다.

똑똑한 에러 검출

느린 비즈니스 로직, 느린 쿼리 질의등을 실시간으로 탐지하고 수집합니다. 수집한 에러들을 이메일, 문자 메시지, 메신저, 웹훅 등에 연동하면 실시간으로 에러를 보고 받을 수 있습니다.

레퍼런스

대한민국 유수의 대기업, 공공기관, 금융권에서 사용하고 있습니다.

뛰어난 유지보수성

관심사 분리

웹 애플리케이션에서 크게 두가지 측면인 View(뷰)와 데이터(API)가 논리적/물리적으로 분리되어 있으며 모든 데이터는 HTTP/JSON 기반으로 전송됩니다.

계층화된 구조

비즈니스 로직들이 API Endpoint -> Service -> Repository 계층으로 순으로 처리되며 각 계층에서 필요한 처리만 담당하기 때문에 트랜잭션 경계지점이 명확하고 코드 유지보수성이 뛰어납니다.

무중단 배포

AXBoot Jenkins와 AXBoot Ansible을 함께 사용하면 AXBoot로 만들어진 애플리케이션을 하루에도 수십번 서비스 중단 없이 통합/배포 할 수 있습니다.
